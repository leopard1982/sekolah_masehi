Tag Menampilkan Informasi

Untuk menampilkan informasi dalam HTML ada beberapa tag yang bisa kita pakai, antara lain:

  • <p></p> dipakai untuk menampilkan data dalam bentuk paragraf
  • <span></span> dipakai untuk menampilkan tulisan yang bisa tempelkan ke dalam paragraf
  • <pre></pre> dipakai untuk menampilkan dalam bentuk terformat kita bisa melihat seperti huruf jenis courier new
  • <marquee></marquee> dipakai untuk menampilkan huruf berjalan
  • <blockquote></blockquote> dipakai untuk menampilkan kutipan teks dari sumber lain
  • <footer></footer> dipakai untuk menampilkan catatan kaki (biasa ada di bawah)
  • <br/> dipakai untuk membuat ganti baris (atau seperti kita menekan tombol enter)
  • <hr/> dipakai untuk menampilkan garis panjang
  • <img src='lokasi'/> dipakai untuk menampilkan gambar. untuk menampilkan gambar ini kita harus menentukan source nya dengan mengetikkan src='nama file' di dalam tag img tersebut
